Тема работы

Разработка языкового процессора для анализа комментариев в PHP. Распознавание однострочных (//, #) и многострочных (/*...*/) комментариев.

Особенности синтаксиса

Комментарии в PHP игнорируются интерпретатором и используются для аннотации кода. Основные типы:

// Однострочный комментарий
# Альтернативный однострочный
/* Многострочный 
   комментарий */

Примеры верных комментариев

Согласно автоматной грамматике, корректными считаются:

// Простой комментарий
# Комментарий с кавычкой \"
/* Многострочный 
   комментарий */
echo "Текст"; // Комментарий после кода
/**/  // Пустой многострочный комментарий
