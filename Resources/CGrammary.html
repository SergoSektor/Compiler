Классификация грамматики комментариев PHP

Разработанная грамматика G[PHP_Comments_FA] относится к классу автоматных (регулярных) грамматик по классификации Хомского. Формально это определяется следующими свойствами:

Формальное определение

Грамматика задаётся как кортеж: G = (VT, VN, P, S), где:

  • VT = { /, *, #, ␤, c } - терминалы
  • VN = { q0, q1, qSL, qML, q*, qa } - нетерминалы
  • S = q0 - стартовый символ
  • Правила P имеют автоматную структуру

Критерии автоматной грамматики

Все продукции грамматики соответствуют форме:

A → aB | a | ε

где:

  • A, B ∈ VN (состояния автомата)
  • a ∈ VT (входные символы)
  • ε - пустая строка (конечное состояние)

Примеры продукций

q0 → / q1       // Переход между состояниями
qSL → ␤ qa     // Завершение однострочного комментария
q* → / qa       // Завершение многострочного комментария
qa → ε               // Конечное состояние

Особенности реализации

  1. Все правила праворекурсивны
  2. Нет продукции с левой рекурсией
  3. Максимальная длина правой части - 2 символа
  4. Используется только один нетерминал в левой части

Вывод

Грамматика комментариев PHP удовлетворяет всем требованиям автоматных грамматик:

  • Может быть реализована через конечный автомат
  • Допускает обработку регулярными выражениями
  • Соответствует ограничениям типа 3 по Хомскому
