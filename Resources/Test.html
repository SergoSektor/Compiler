Тестовые примеры для комментариев PHP

✅ Корректные комментарии

1. Однострочный комментарий (//)

// Однострочный комментарий

2. Альтернативный однострочный (#)

# Ещё один однострочный комментарий

3. Многострочный комментарий

/* Многострочный
   комментарий */

4. Пустой многострочный комментарий

/**/

❌ Некорректные комментарии

1. Неполный однострочный комментарий

/ Однострочный комментарий

Ошибка: отсутствует второй символ '/'

2. Незакрытый многострочный комментарий

/* Многострочный
   комментарий *

Ошибка: отсутствует закрывающий '*/'
