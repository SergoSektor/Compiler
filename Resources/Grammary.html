Автоматная грамматика комментариев PHP

G[PHP_Comments_FA] = (VT, VN, P, S):
VT = { '/', '*', '#', ␤, c }
VN = { q0, q1, qSL, qML, q*, qa }
S = q0
P = {
 1. q0 → / q1
 2. q0 → # qSL
 3. q1 → / qSL
 4. q1 → * qML
 5. qSL → c qSL
 6. qSL → ␤ qa
 7. qML → c qML
 8. qML → * q*
 9. q* → * q*
 10. q* → c qML
 11. q* → / qa
 12. qa → ε
}

Обозначения:

  • ␤ - символ перевода строки (U+2424)
  • c - любой символ, кроме '/', '*', '#', ␤
  • ε - пустая строка (конечное состояние)
